Compiling, Linking, Flashing
=============================

Sphinx
------

`Sphinx GitHub <https://github.com/sphinx-doc/sphinx>`_

1. **Install Sphinx**

    .. code-block:: bash

        pip install sphinx sphinx-book-theme

2. **Build the Documentation**

    .. code-block:: bash

        make html

3. **View Locally**

    .. code-block:: bash

        python -m http.server 8080 --directory docs/html
